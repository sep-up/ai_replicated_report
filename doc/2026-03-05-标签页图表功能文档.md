# 标签页图表功能文档

## 1. 功能概述

本功能实现智能制造报表看板中间区域的标签页图表，包含两个标签页：
1. **本月工单产出趋势**：展示本月每日工单产出数量的变化趋势
2. **本月报工进度趋势**：展示本月每日报工进度的变化趋势

## 3. 图表实现方案

### 3.1 本月工单产出趋势图表

- **图表类型**：柱状图
- **数据展示**：
  - X轴：日期（本月每日，格式：MM-DD）
  - Y轴：工单数量
- **图表配置**：
  - 标题：本月工单产出趋势
  - 图例：工单数量
  - 颜色：蓝色渐变 (#409EFF 到 #66B1FF)
  - 动画：平滑过渡
  - 响应式：适应不同屏幕尺寸
- **交互功能**：
  - 鼠标悬停显示详细数据
  - 点击图例切换数据显示/隐藏

### 3.2 本月报工进度趋势图表

- **图表类型**：柱状图
- **数据展示**：
  - X轴：日期（本月每日，格式：MM-DD）
  - Y轴：报工数量
- **图表配置**：
  - 标题：本月报工进度趋势
  - 图例：报工数量
  - 颜色：绿色渐变 (#67C23A 到 #85CE61)
  - 动画：平滑过渡
  - 响应式：适应不同屏幕尺寸
- **交互功能**：
  - 鼠标悬停显示详细数据
  - 点击图例切换数据显示/隐藏

## 4. 标签页实现

### 4.1 UI组件

- 使用自定义标签页组件，样式参考 Radix Vue 标签页
- 标签页数量：2个
- 标签页名称：
  1. 本月工单产出趋势
  2. 本月报工进度趋势
- 标签页样式：
  - 背景色：bg-muted
  - 文字颜色：text-muted-foreground
  - 选中状态：白色背景，深色文字，带阴影效果

### 4.2 交互逻辑

- 默认显示第一个标签页（工单产出趋势）
- 点击标签页切换图表内容
- 标签页切换时重新请求对应 API 数据
- 数据加载时显示加载动画
- 数据加载失败时显示错误提示

## 5. 技术实现

### 5.1 前端技术栈

- 框架：Vue.js 2.x
- 图表库：ECharts
- UI组件库：Element UI
- HTTP请求：Axios

### 5.2 后端API

- 接口路径：`/jy-app/api/product/work/reports/findWorkOrderCountByMonth`
- 接口路径：`/jy-app/api/product/work/reports/findTaskProgressByMonth`

### 5.3 实现步骤

1. **创建API模块**：在 `src/api` 目录下创建 `product.js` 文件，添加API接口方法
2. **创建图表组件**：在 `src/components` 目录下创建 `ChartTabs` 组件
3. **实现图表渲染**：使用 ECharts 实现折线图渲染
4. **集成到主页面**：将图表组件集成到 `overview` 页面

### 6.2 图表组件

```vue
<template>
  <div class="chart-tabs">
    <div 
      role="tablist" 
      aria-orientation="horizontal" 
      tabindex="0" 
      data-orientation="horizontal" 
      dir="ltr" 
      class="bg-muted text-muted-foreground inline-flex h-9 items-center justify-center rounded-lg p-1"
    >
      <button 
        id="radix-vue-tabs-v-0-trigger-trends" 
        role="tab" 
        type="button" 
        :aria-selected="activeTab === 'workOrder'" 
        :aria-controls="activeTab === 'workOrder' ? 'radix-vue-tabs-v-0-content-trends' : ''" 
        :data-state="activeTab === 'workOrder' ? 'active' : 'inactive'" 
        data-orientation="horizontal" 
        :tabindex="activeTab === 'workOrder' ? '0' : '-1'" 
        :class="activeTab === 'workOrder' ? 'data-[state=active]:bg-background data-[state=active]:text-foreground' : ''"
        class="ring-offset-background focus-visible:ring-ring data-[state=active]:bg-background data-[state=active]:text-foreground inline-flex items-center justify-center whitespace-nowrap rounded-md px-3 py-1 text-sm font-medium transition-all focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-offset-2 disabled:pointer-events-none disabled:opacity-50 data-[state=active]:shadow"
        @click="activeTab = 'workOrder'; handleTabClick('workOrder')"
      >
        本月工单产出趋势
      </button>
      <button 
        id="radix-vue-tabs-v-0-trigger-visits" 
        role="tab" 
        type="button" 
        :aria-selected="activeTab === 'taskProgress'" 
        :aria-controls="activeTab === 'taskProgress' ? 'radix-vue-tabs-v-0-content-visits' : ''" 
        :data-state="activeTab === 'taskProgress' ? 'active' : 'inactive'" 
        data-orientation="horizontal" 
        :tabindex="activeTab === 'taskProgress' ? '0' : '-1'" 
        :class="activeTab === 'taskProgress' ? 'data-[state=active]:bg-background data-[state=active]:text-foreground' : ''"
        class="ring-offset-background focus-visible:ring-ring data-[state=active]:bg-background data-[state=active]:text-foreground inline-flex items-center justify-center whitespace-nowrap rounded-md px-3 py-1 text-sm font-medium transition-all focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-offset-2 disabled:pointer-events-none disabled:opacity-50 data-[state=active]:shadow"
        @click="activeTab = 'taskProgress'; handleTabClick('taskProgress')"
      >
        本月报工进度趋势
      </button>
    </div>
    
    <div v-loading="loading" class="chart-container">
      <div ref="chartContainer" class="chart"></div>
    </div>
  </div>
</template>

<script>
import * as echarts from 'echarts'
import { getWorkOrderCountByMonth, getTaskProgressByMonth } from '@/api/report'

export default {
  name: 'ChartTabs',
  data() {
    return {
      activeTab: 'workOrder',
      loading: false,
      chart: null,
      chartData: {
        workOrder: null,
        taskProgress: null
      }
    }
  },
  mounted() {
    this.loadChartData()
    window.addEventListener('resize', this.handleResize)
  },
  beforeDestroy() {
    if (this.chart) {
      this.chart.dispose()
    }
    window.removeEventListener('resize', this.handleResize)
  },
  methods: {
    handleResize() {
      if (this.chart) {
        this.chart.resize()
      }
    },
    handleTabClick(tabName) {
      this.activeTab = tabName
      this.renderChart()
    },
    async loadChartData() {
      this.loading = true
      try {
        const [workOrderRes, taskProgressRes] = await Promise.all([
          getWorkOrderCountByMonth(),
          getTaskProgressByMonth()
        ])

        if (workOrderRes.data.code === 200) {
          const data = workOrderRes.data.data || []
          this.chartData.workOrder = {
            dates: data.map(item => item.name),
            counts: data.map(item => item.ext1 || item.quantity || 0)
          }
        }

        if (taskProgressRes.data.code === 200) {
          const data = taskProgressRes.data.data || []
          this.chartData.taskProgress = {
            dates: data.map(item => item.name),
            counts: data.map(item => item.ext1 || item.quantity || 0)
          }
        }

        this.$nextTick(() => {
          this.renderChart()
        })
      } catch (error) {
        console.error('获取图表数据失败:', error)
        this.$notify({
          title: '错误',
          message: '获取图表数据失败',
          type: 'error',
          duration: 2000
        })
      } finally {
        this.loading = false
      }
    },
    renderChart() {
      if (!this.$refs.chartContainer) return

      if (this.chart) {
        this.chart.dispose()
      }

      this.chart = echarts.init(this.$refs.chartContainer)
      
      const chartData = this.chartData[this.activeTab]
      if (!chartData) return

      const isWorkOrder = this.activeTab === 'workOrder'
      
      this.chart.setOption({
        tooltip: {
          trigger: 'axis',
          formatter: '{b}: {c} 单'
        },
        grid: {
          left: '3%',
          right: '4%',
          bottom: '15%',
          top: '10%',
          containLabel: true
        },
        xAxis: {
          type: 'category',
          data: chartData.dates,
          axisLabel: {
            rotate: 45,
            interval: 0,
            fontSize: 11
          }
        },
        yAxis: {
          type: 'value',
          name: isWorkOrder ? '工单数量' : '报工数量',
          axisLabel: {
            fontSize: 11
          }
        },
        series: [{
          name: isWorkOrder ? '工单数量' : '报工数量',
          data: chartData.counts,
          type: 'bar',
          barWidth: '60%',
          itemStyle: {
            color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
              { offset: 0, color: isWorkOrder ? '#1890FF' : '#52C41A' },
              { offset: 1, color: isWorkOrder ? '#69C0FF' : '#95DE64' }
            ])
          }
        }]
      })
    }
  }
}
</script>
```

## 7. 注意事项

1. **API接口**：确保后端API接口已部署并可正常访问
2. **数据格式**：确保API返回的数据格式与文档描述一致
3. **权限控制**：确保用户有访问这些API的权限
4. **错误处理**：添加适当的错误处理逻辑，提高用户体验
5. **性能优化**：考虑数据缓存策略，避免频繁请求API
6. **兼容性**：确保图表在不同浏览器中正常显示
7. **响应式设计**：确保图表在不同屏幕尺寸下都能正常显示
