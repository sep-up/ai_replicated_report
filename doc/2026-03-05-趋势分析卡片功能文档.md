## 1. 功能概述

### 1.1 功能描述

本功能模块位于智能制造报表看板的下方区域，包含三个趋势分析卡片：

1. **近7天生产订单数量**：展示最近7天有效工单状态统计，包括工单计划数、在制工单数、已完成工单数
2. **近7日生产工序数量**：展示最近7天有效生产工序状态统计
3. **近7天质检不良品数量**：展示最近七天的质检缺陷品统计

### 1.2 布局设计

- 三列等宽布局，响应式适配
- 每个卡片包含：
  - 卡片标题
  - 图表展示区域
  - 数据图例（如适用）

## 3. 图表实现方案

### 3.1 近7天生产订单数量（折线图）

**图表类型**：折线图

**配置参数**：
- X轴：固定类别（工单计划数、在制工单数、已完成工单数）
- Y轴：订单数量
- 颜色：使用主色调蓝色系
- 图表大小：响应式，根据卡片宽度自适应
- 交互：鼠标悬停显示具体数值

**ECharts配置示例**：

```javascript
{
  tooltip: {
    trigger: 'axis'
  },
  grid: {
    left: '3%',
    right: '4%',
    bottom: '3%',
    containLabel: true
  },
  xAxis: {
    type: 'category',
    data: ['工单计划数', '在制工单数', '已完成工单数']
  },
  yAxis: {
    type: 'value'
  },
  series: [{
    data: [156, 89, 42],
    type: 'line',
    smooth: true,
    itemStyle: {
      color: '#409EFF'
    },
    areaStyle: {
      color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
        { offset: 0, color: 'rgba(64, 158, 255, 0.3)' },
        { offset: 1, color: 'rgba(64, 158, 255, 0.1)' }
      ])
    }
  }]
}
```

**图表类型**：环形图

**配置参数**：
- 展示各工序的工单数量占比

- 图例：底部显示工序名称和数量

- 交互：鼠标悬停显示具体数值和占比

## 4. 数据处理逻辑

### 4.1 数据加载与缓存

1. 页面初始化时调用三个API接口获取数据
2. 实现数据缓存机制，避免重复请求
3. 支持手动刷新按钮（可选）
4. 自动刷新：每5分钟刷新一次（可配置）

### 4.2 数据转换逻辑

**近7天生产订单数量**：
- 从接口返回的data数组中提取name作为categories
- 提取ext1或quantity作为counts
- 确保数据顺序与接口返回一致

**近7日生产工序数量**：
- 从接口返回的data数组中提取name和quantity
- 转换为{name, value}格式
- 计算总数：sum(value)
- 按value降序排序，确保占比大的工序显示在前

**近7天质检不良数量**：
- 从接口返回的data数组中提取name
- 提取ext1或quantity作为value
- 转换为{name, value}格式
- 计算总数：sum(value)
- 按value降序排序，确保占比大的不良类型显示在前

## 5. 技术实现

### 5.1 前端实现

**API调用封装**：

```javascript
// src/api/product.js
import request from '@/utils/request'

// 获取近7天生产订单数量
export function getWorkOrderBySevenDays() {
  return request({
    url: '/jy-app/api/product/work/reports/findWorkOrderBySevenDays',
    method: 'get'
  })
}

// 获取近7日生产工序数量
export function getProductProducesBySevenDays() {
  return request({
    url: '/jy-app/api/product/reports/findProductProducesBySevenDays',
    method: 'get'
  })
}

// 获取近7天质检不良数量
export function getDefectStaticBySevenDaysInStatus() {
  return request({
    url: '/jy-app/api/product/work/reports/inspection/defectStaticBySevenDaysInStatus',
    method: 'get'
  })
}
```

**组件实现**：

```vue
<!-- src/components/TrendAnalysisCards/index.vue -->
<template>
  <div class="trend-analysis-cards">
    <!-- 近7天生产订单数量 -->
    <div class="card-item">
      <div class="card-header">
        <span class="card-title">近7天生产订单数量</span>
      </div>
      <div v-loading="loading.order" class="chart-container">
        <div ref="orderChart" class="chart"></div>
      </div>
    </div>

    <!-- 近7日生产工序数量 -->
    <div class="card-item">
      <div class="card-header">
        <span class="card-title">近7日生产工序数量</span>
      </div>
      <div v-loading="loading.process" class="chart-container">
        <div ref="processChart" class="chart"></div>
      </div>
    </div>

    <!-- 近7天质检不良数量 -->
    <div class="card-item">
      <div class="card-header">
        <span class="card-title">近7天质检不良数量</span>
      </div>
      <div v-loading="loading.defect" class="chart-container">
        <div ref="defectChart" class="chart"></div>
      </div>
    </div>
  </div>
</template>

<script>
import * as echarts from 'echarts'
import {
  getWorkOrderBySevenDays,
  getProductProducesBySevenDays,
  getDefectStaticBySevenDaysInStatus
} from '@/api/report'

export default {
  name: 'TrendAnalysisCards',
  data() {
    return {
      loading: {
        order: false,
        process: false,
        defect: false
      },
      orderChart: null,
      processChart: null,
      defectChart: null,
      orderData: {
        categories: [],
        counts: []
      },
      processData: [],
      defectData: []
    }
  },
  mounted() {
    this.initCharts()
    this.loadData()
    window.addEventListener('resize', this.handleResize)
  },
  beforeDestroy() {
    if (this.orderChart) {
      this.orderChart.dispose()
    }
    if (this.processChart) {
      this.processChart.dispose()
    }
    if (this.defectChart) {
      this.defectChart.dispose()
    }
    window.removeEventListener('resize', this.handleResize)
  },
  methods: {
    initCharts() {
      this.orderChart = echarts.init(this.$refs.orderChart)
      this.processChart = echarts.init(this.$refs.processChart)
      this.defectChart = echarts.init(this.$refs.defectChart)
    },
    handleResize() {
      if (this.orderChart) {
        this.orderChart.resize()
      }
      if (this.processChart) {
        this.processChart.resize()
      }
      if (this.defectChart) {
        this.defectChart.resize()
      }
    },
    async loadData() {
      try {
        await Promise.all([
          this.loadOrderData(),
          this.loadProcessData(),
          this.loadDefectData()
        ])
      } catch (error) {
        console.error('加载数据失败:', error)
      }
    },
    async loadOrderData() {
      this.loading.order = true
      try {
        const res = await getWorkOrderBySevenDays()
        if (res.data.code === 200) {
          const data = res.data.data || []
          this.orderData = {
            categories: data.map(item => item.name),
            counts: data.map(item => item.ext1 || item.quantity || 0)
          }
          this.updateOrderChart()
        }
      } catch (error) {
        console.error('获取生产订单数据失败:', error)
        this.$notify({
          title: '错误',
          message: '获取生产订单数据失败',
          type: 'error',
          duration: 2000
        })
      } finally {
        this.loading.order = false
      }
    },
    async loadProcessData() {
      this.loading.process = true
      try {
        const res = await getProductProducesBySevenDays()
        if (res.data.code === 200) {
          this.processData = (res.data.data || []).map(item => ({
            name: item.name,
            value: item.quantity || 0
          }))
          this.updateProcessChart()
        }
      } catch (error) {
        console.error('获取生产工序数据失败:', error)
        this.$notify({
          title: '错误',
          message: '获取生产工序数据失败',
          type: 'error',
          duration: 2000
        })
      } finally {
        this.loading.process = false
      }
    },
    async loadDefectData() {
      this.loading.defect = true
      try {
        const res = await getDefectStaticBySevenDaysInStatus()
        if (res.data.code === 200) {
          this.defectData = (res.data.data || []).map(item => ({
            name: item.name,
            value: item.ext1 || item.quantity || 0
          }))
          this.updateDefectChart()
        }
      } catch (error) {
        console.error('获取质检不良数据失败:', error)
        this.$notify({
          title: '错误',
          message: '获取质检不良数据失败',
          type: 'error',
          duration: 2000
        })
      } finally {
        this.loading.defect = false
      }
    },
    updateOrderChart() {
      const option = {
        tooltip: {
          trigger: 'axis'
        },
        grid: {
          left: '3%',
          right: '4%',
          bottom: '15%',
          top: '10%',
          containLabel: true
        },
        xAxis: {
          type: 'category',
          data: this.orderData.categories,
          axisLabel: {
            rotate: 30,
            interval: 0,
            fontSize: 11
          }
        },
        yAxis: {
          type: 'value',
          axisLabel: {
            fontSize: 11
          }
        },
        series: [{
          data: this.orderData.counts,
          type: 'line',
          smooth: true,
          itemStyle: {
            color: '#409EFF'
          },
          areaStyle: {
            color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
              { offset: 0, color: 'rgba(64, 158, 255, 0.3)' },
              { offset: 1, color: 'rgba(64, 158, 255, 0.1)' }
            ])
          }
        }]
      }
      this.orderChart.setOption(option)
    },
    updateProcessChart() {
      const total = this.processData.reduce((sum, item) => sum + item.value, 0)
      const option = {
        tooltip: {
          trigger: 'item',
          formatter: '{a} <br/>{b}: {c} ({d}%)'
        },
        legend: {
          orient: 'horizontal',
          bottom: 10,
          left: 'center',
          formatter: (name) => {
            const item = this.processData.find(item => item.name === name)
            return item ? `${name}: ${item.value}` : name
          },
          textStyle: {
            fontSize: 11
          }
        },
        series: [{
          name: '工序数量',
          type: 'pie',
          radius: ['40%', '70%'],
          center: ['50%', '40%'],
          avoidLabelOverlap: false,
          label: {
            show: true,
            position: 'center',
            formatter: `总数\n${total}`,
            fontSize: 14,
            fontWeight: 'bold'
          },
          emphasis: {
            label: {
              show: true,
              fontSize: '16',
              fontWeight: 'bold'
            }
          },
          labelLine: {
            show: false
          },
          itemStyle: {
            borderRadius: 8
          },
          data: this.processData
        }]
      }
      this.processChart.setOption(option)
    },
    updateDefectChart() {
      const option = {
        tooltip: {
          trigger: 'item',
          formatter: '{a} <br/>{b}: {c} ({d}%)'
        },
        legend: {
          orient: 'horizontal',
          bottom: 10,
          left: 'center',
          formatter: (name) => {
            const item = this.defectData.find(item => item.name === name)
            return item ? `${name}: ${item.value}` : name
          },
          textStyle: {
            fontSize: 11
          }
        },
        series: [{
          name: '不良类型',
          type: 'pie',
          radius: '60%',
          center: ['50%', '40%'],
          data: this.defectData,
          emphasis: {
            itemStyle: {
              shadowBlur: 10,
              shadowOffsetX: 0,
              shadowColor: 'rgba(0, 0, 0, 0.5)'
            }
          },
          label: {
            fontSize: 11
          }
        }]
      }
      this.defectChart.setOption(option)
    }
  }
}
</script>
```
