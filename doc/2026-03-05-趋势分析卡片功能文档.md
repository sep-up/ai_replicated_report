## 1. 功能概述

### 1.1 功能描述

本功能模块位于智能制造报表看板的下方区域，包含三个趋势分析卡片：

1. **近7天生产订单数量**：展示最近7天有效工单状态统计，包括工单计划数、在制工单数、已完成工单数
2. **近7日生产工序数量**：展示最近7天有效生产工序状态统计
3. **近7天质检不良品数量**：展示最近七天的质检缺陷品统计

### 1.2 布局设计

- 三列等宽布局，响应式适配
- 每个卡片包含：
  - 卡片标题
  - 图表展示区域
  - 数据图例（如适用）

## 3. 图表实现方案

### 3.1 近7天生产订单数量（折线图）

**图表类型**：折线图

**配置参数**：
- X轴：固定类别（工单计划数、在制工单数、已完成工单数）
- Y轴：订单数量
- 颜色：使用主色调蓝色系
- 图表大小：响应式，根据卡片宽度自适应
- 交互：鼠标悬停显示具体数值

**ECharts配置示例**：

```javascript
{
  tooltip: {
    trigger: 'axis'
  },
  grid: {
    left: '3%',
    right: '4%',
    bottom: '3%',
    containLabel: true
  },
  xAxis: {
    type: 'category',
    data: ['工单计划数', '在制工单数', '已完成工单数']
  },
  yAxis: {
    type: 'value'
  },
  series: [{
    data: [156, 89, 42],
    type: 'line',
    smooth: true,
    itemStyle: {
      color: '#409EFF'
    },
    areaStyle: {
      color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
        { offset: 0, color: 'rgba(64, 158, 255, 0.3)' },
        { offset: 1, color: 'rgba(64, 158, 255, 0.1)' }
      ])
    }
  }]
}
```

### 3.2 近7日生产工序数量（环形图）

**图表类型**：环形图

**配置参数**：
- 展示各工序的工单数量占比
- 中心显示总数
- 图例：右侧显示工序名称和数量
- 颜色：使用ECharts默认配色方案
- 交互：鼠标悬停显示具体数值和占比

**ECharts配置示例**：

```javascript
{
  tooltip: {
    trigger: 'item',
    formatter: '{a} <br/>{b}: {c} ({d}%)'
  },
  legend: {
    orient: 'vertical',
    right: 10,
    top: 'center',
    formatter: function(name) {
      var item = data.find(item => item.name === name);
      return name + ': ' + item.value;
    }
  },
  series: [{
```

## 4. 数据处理逻辑

### 4.1 数据加载与缓存

1. 页面初始化时调用三个API接口获取数据
2. 实现数据缓存机制，避免重复请求
3. 支持手动刷新按钮（可选）
4. 自动刷新：每5分钟刷新一次（可配置）

### 4.2 数据转换逻辑

**近7天生产订单数量**：
- 从接口返回的data数组中提取name作为categories
- 提取ext1或quantity作为counts
- 确保数据顺序与接口返回一致

**近7日生产工序数量**：
- 从接口返回的data数组中提取name和quantity
- 转换为{name, value}格式
- 计算总数：sum(value)
- 按value降序排序，确保占比大的工序显示在前

**近7天质检不良数量**：
- 从接口返回的data数组中提取name
- 提取ext1或quantity作为value
- 转换为{name, value}格式
- 计算总数：sum(value)
- 按value降序排序，确保占比大的不良类型显示在前

## 5. 技术实现

### 5.1 前端实现

**API调用封装**：

```javascript
// src/api/product.js
import request from '@/utils/request'

// 获取近7天生产订单数量
export function getWorkOrderBySevenDays() {
  return request({
    url: '/jy-app/api/product/work/reports/findWorkOrderBySevenDays',
    method: 'get'
  })
}

// 获取近7日生产工序数量
export function getProductProducesBySevenDays() {
  return request({
    url: '/jy-app/api/product/reports/findProductProducesBySevenDays',
    method: 'get'
  })
}

// 获取近7天质检不良数量
export function getDefectStaticBySevenDaysInStatus() {
  return request({
    url: '/jy-app/api/product/work/reports/inspection/defectStaticBySevenDaysInStatus',
    method: 'get'
  })
}
```

**组件实现**：

```vue
<!-- src/components/TrendAnalysisCards/index.vue -->
<template>
  <div class="trend-analysis-cards">
    <!-- 近7天生产订单数量 -->
    <el-card class="card-item">
      <template slot="header">
        <div class="card-header">
          <span>近7天生产订单数量</span>
          <el-button size="mini" type="primary" plain @click="refreshData">刷新</el-button>
        </div>
      </template>
      <div ref="orderChart" class="chart-container"></div>
    </el-card>

    <!-- 近7日生产工序数量 -->
    <el-card class="card-item">
      <template slot="header">
        <div class="card-header">
          <span>近7日生产工序数量</span>
        </div>
      </template>
      <div ref="processChart" class="chart-container"></div>
    </el-card>

    <!-- 近7天质检不良数量 -->
    <el-card class="card-item">
      <template slot="header">
        <div class="card-header">
          <span>近7天质检不良数量</span>
        </div>
      </template>
      <div ref="defectChart" class="chart-container"></div>
    </el-card>
  </div>
</template>

<script>
import * as echarts from 'echarts'
import { getWorkOrderBySevenDays, getProductProducesBySevenDays, getDefectStaticBySevenDaysInStatus } from '@/api/product'

export default {
  name: 'TrendAnalysisCards',
  data() {
    return {
      orderChart: null,
      processChart: null,
      defectChart: null,
      orderData: {
        categories: [],
        counts: []
      },
      processData: [],
      defectData: []
    }
  },
  mounted() {
    this.initCharts()
    this.loadData()
    // 自动刷新
    this.refreshInterval = setInterval(() => {
      this.loadData()
    }, 5 * 60 * 1000)
  },
  beforeDestroy() {
    if (this.refreshInterval) {
      clearInterval(this.refreshInterval)
    }
    if (this.orderChart) {
      this.orderChart.dispose()
    }
    if (this.processChart) {
      this.processChart.dispose()
    }
    if (this.defectChart) {
      this.defectChart.dispose()
    }
  },
  methods: {
    initCharts() {
      this.orderChart = echarts.init(this.$refs.orderChart)
      this.processChart = echarts.init(this.$refs.processChart)
      this.defectChart = echarts.init(this.$refs.defectChart)
      
      // 响应式
      window.addEventListener('resize', () => {
        this.orderChart.resize()
        this.processChart.resize()
        this.defectChart.resize()
      })
    },
    async loadData() {
      try {
        // 并行请求数据
        const [orderRes, processRes, defectRes] = await Promise.all([
          getWorkOrderBySevenDays(),
          getProductProducesBySevenDays(),
          getDefectStaticBySevenDaysInStatus()
        ])
        
        if (orderRes.data.code === 200) {
          const orderData = orderRes.data.data
          this.orderData = {
            categories: orderData.map(item => item.name),
            counts: orderData.map(item => item.ext1 || item.quantity)
          }
          this.updateOrderChart()
        }
        
        if (processRes.data.code === 200) {
          this.processData = processRes.data.data.map(item => ({
            name: item.name,
            value: item.quantity
          }))
          this.updateProcessChart()
        }
        
        if (defectRes.data.code === 200) {
          this.defectData = defectRes.data.data.map(item => ({
            name: item.name,
            value: item.ext1 || item.quantity
          }))
          this.updateDefectChart()
        }
      } catch (error) {
        console.error('加载数据失败:', error)
      }
    },
    refreshData() {
      this.loadData()
    },
    updateOrderChart() {
      const option = {
        tooltip: {
          trigger: 'axis'
        },
        grid: {
          left: '3%',
          right: '4%',
          bottom: '3%',
          containLabel: true
        },
        xAxis: {
          type: 'category',
          data: this.orderData.categories
        },
        yAxis: {
          type: 'value'
        },
        series: [{
          data: this.orderData.counts,
          type: 'line',
          smooth: true,
          itemStyle: {
            color: '#409EFF'
          },
          areaStyle: {
            color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
              { offset: 0, color: 'rgba(64, 158, 255, 0.3)' },
              { offset: 1, color: 'rgba(64, 158, 255, 0.1)' }
            ])
          }
        }]
      }
      this.orderChart.setOption(option)
    },
    updateProcessChart() {
      const total = this.processData.reduce((sum, item) => sum + item.value, 0)
      const option = {
        tooltip: {
          trigger: 'item',
          formatter: '{a} <br/>{b}: {c} ({d}%)'
        },
        legend: {
          orient: 'vertical',
          right: 10,
          top: 'center',
          formatter: (name) => {
            const item = this.processData.find(item => item.name === name)
            return name + ': ' + item.value
          }
        },
        series: [{
          name: '工序数量',
          type: 'pie',
          radius: ['40%', '70%'],
          avoidLabelOverlap: false,
          label: {
            show: true,
            formatter: `总数: ${total}\n{b}\n{d}%`
          },
          emphasis: {
            label: {
              show: true,
              fontSize: '18',
              fontWeight: 'bold'
            }
          },
          labelLine: {
            show: false
          },
          data: this.processData
        }]
      }
      this.processChart.setOption(option)
    },
    updateDefectChart() {
      const option = {
        tooltip: {
          trigger: 'item',
          formatter: '{a} <br/>{b}: {c} ({d}%)'
        },
        legend: {
          orient: 'vertical',
          right: 10,
          top: 'center',
          formatter: (name) => {
            const item = this.defectData.find(item => item.name === name)
            return name + ': ' + item.value
          }
        },
        series: [{
          name: '不良类型',
          type: 'pie',
          radius: '60%',
          center: ['40%', '50%'],
          data: this.defectData,
          emphasis: {
            itemStyle: {
              shadowBlur: 10,
              shadowOffsetX: 0,
              shadowColor: 'rgba(0, 0, 0, 0.5)'
            }
          }
        }]
      }
      this.defectChart.setOption(option)
    }
  }
}
</script>
```
